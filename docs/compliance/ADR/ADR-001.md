# ADR-001: SD-JWT VC as Credential Format (E-10)

**Date:** 2026-03-06
**Status:** Accepted
**Deciders:** Jonas (Late-bloomer420)

---

## Context

The EUDI Wallet Regulation (eIDAS 2.0) and the associated Commission Implementing Regulations (CIR 2024/2977, 2024/2979, 2024/2982) mandate interoperability across EU member states. A common credential format is required for PID (Person Identification Data) and EAA (Electronic Attestation of Attributes).

Candidates evaluated:

| Format | Selective Disclosure | W3C VC | EUDI mandate | Notes |
|--------|---------------------|---------|--------------|-------|
| W3C VC (JWT) | No | Yes | No | No SD; not in ARF |
| SD-JWT VC (draft-ietf-oauth-sd-jwt-vc-11) | Yes | Partial | Yes | EUDI Architecture Reference Framework §6 |
| ISO mdoc (ISO 18013-5) | Yes | No | Yes | Mobile-only; BLE/NFC transport |
| AnonCreds | Yes | No | No | Hyperledger-specific; not in eIDAS scope |

---

## Decision

Adopt **SD-JWT VC** (`draft-ietf-oauth-sd-jwt-vc-11`) as the primary credential format for miTch.

Implementation: `src/packages/shared-crypto/src/sd-jwt-vc.ts`

Key commitments:
- `vct` claim identifies credential type (MUST be URI per CIR 2024/2977)
- `_sd_alg: sha-256` always present
- `cnf.jwk` for Key Binding (holder public key)
- `kb+jwt` typed Key Binding JWT for presentation
- `typ: vc+sd-jwt` in JOSE header
- `iss` validated as URI (issuer authority identifier)

---

## Consequences

**Positive:**
- EUDI ARF compliant — interoperable with any EU wallet implementing SD-JWT VC
- Selective disclosure: holder can reveal individual claims without exposing full credential
- Standard JOSE tooling (`jose` library) — no custom crypto required
- `cnf.jwk` enables unlinkable Key Binding across presentations

**Negative / Trade-offs:**
- ISO mdoc not supported — mobile-proximity (BLE/NFC) presentations out of scope for v1
- W3C VC JSON-LD context not maintained — non-linked-data consumers may need adaptation
- `status` claim requires a deployed status list endpoint for revocation (currently 🟡 partial)
